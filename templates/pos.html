Modal Templates

Modal Templates are embedded in the head of the html to be compiled with Underscore. They should include:

  • a button with name="ok" and data-response="SOME_URL_HERE"
  • a button with name="cancel"
  • a number of fields each with a data-property="SOME_FIELD_NAME"

Any select fields with from-cache="STORE_LOCATION3" will have their innerhtml replaced with values fetched from the server.

When the "ok" button is clicked, all input and select fields will be gathered along with the data-response value of the button and passed as an object to the callback provided to the ModalTemplate. It will look something like:

{
	url: 'somewhere.html',
	values: {
		FIELD_A: 'asdf',
		FIELD_B: 3
	}
}
